# Mugsy's Mugs — Artisan Ceramics ☕🏺

**RO:** O pagină premium de e-commerce pentru un brand fictiv de ceramică artizanală, construită ca experiență vizuală interactivă, bilingvă și responsive.

**EN:** A premium e-commerce landing page for a fictional artisan ceramics brand, built as an interactive, bilingual, responsive shopping experience.

---

## 🇷🇴 Descriere

**Mugsy's Mugs** prezintă o colecție de 8 produse ceramice într-un layout editorial, cu imagini mari, carduri premium, micro-interacțiuni și controale clare pentru selecția produselor.

Proiectul este potrivit pentru:

- portofoliu front-end;
- CodePen Challenge / UI experiment;
- demo de magazin artizanal;
- studiu de caz pentru design premium + accesibilitate.

### Funcționalități

- Layout responsive pentru desktop, tabletă și mobil
- Comutare temă Dark / Light
- Comutare limbă EN / RO
- Coș demonstrativ cu counter
- Validare pentru opțiunile obligatorii
- Toast feedback pentru acțiuni
- Sunet opțional prin Web Audio API
- Efecte vizuale: cursor glow, ripple, card flash, animații subtile
- Respect pentru `prefers-reduced-motion`
- Link rapid către versiunea V2

### Decizii UI/UX

Interfața folosește o direcție vizuală editorială, cu tipografie contrastantă, imagini full-card și accente aurii. Textul de pe carduri rămâne lizibil prin gradient overlay, iar controalele sunt vizibile permanent, nu ascunse doar la hover.

## 🇬🇧 Description

**Mugsy's Mugs** showcases a collection of 8 ceramic products in an editorial-style layout, with large imagery, premium cards, subtle micro-interactions, and clear product selection controls.

This project works well as:

- a front-end portfolio piece;
- a CodePen Challenge / UI experiment;
- an artisan shop demo;
- a case study for premium design + accessibility.

### Features

- Responsive layout for desktop, tablet, and mobile
- Dark / Light theme switcher
- EN / RO language switcher
- Demo cart counter
- Required option validation
- Toast feedback for user actions
- Optional Web Audio API sound
- Visual effects: cursor glow, ripple, card flash, subtle animations
- `prefers-reduced-motion` support
- Quick link to the V2 version

### UI/UX Decisions

The interface follows an editorial premium direction, with strong typography, full-card images, and gold accents. Product card text stays readable through gradient overlays, while controls remain permanently visible instead of being hidden behind hover-only interactions.
